Quickstart guide

Quick-Start

Install Atom

Atom is an IDE for languages including Julia. You will install Julia as the first step of the Atom installation. Install Atom by following the instructions here.

Install PressureFieldContact

Paste the following code into a text file ending in .jl and save it. Open this file in ATOM and then press ctrl + shift + enter to run the file and install PressureFieldContact and the necessary dependencies.

using Pkg
Pkg.add("PressureFieldContact")

Run example

Go to the examples section and run one of the examples.
